# Skills

Place reusable model guidance files in this directory as Markdown (`.md`) files.

Example:

- `code-review.md`
- `security-audit.md`

In the CLI, manage session skills with:

- `/skills list`
- `/skills status`
- `/skills enable <name>`
- `/skills disable <name>`

Use `--skills-dir <path>` to load skills from a different directory.
