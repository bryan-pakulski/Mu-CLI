# Architecture Decision Records (ADRs)

This folder captures durable architecture decisions for `mu_cli`.

## Format

- File naming: `NNNN-title.md` (e.g. `0001-session-boundaries.md`).
- Status values: `proposed`, `accepted`, `superseded`.
- Keep each ADR short and practical:
  - Context
  - Decision
  - Consequences
